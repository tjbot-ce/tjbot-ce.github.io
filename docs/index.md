# TJBot API Documentation

TJBot can be programmed using either:

- The [Node.js TypeScript library](https://github.com/tjbot-ce/node-tjbotlib)
- The [Python library](https://github.com/tjbot-ce/python-tjbotlib)

Both libraries let you access all of TJBot's capabilities, such as listening, speaking, seeing, and shining the LED.

## API docs

- node-tjbotlib TypeScript library: [node-tjbotlib/3.0.0](node-tjbotlib/3.0.0/)
- python-tjbotlib Python library: [python-tjbotlib/3.0.0](python-tjbotlib/3.0.0/)
